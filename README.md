
## 🖥️🛠 SMARTUP

CRM, ERP, HRM Smartup system, distribution automation program, trade, business processes, warehouse automation Uzbekistan, Kazakhstan, Kyrgyzstan, Tajikistan and the CIS.

## Deployment

To deploy this project run

```bash
  npm install
  gulp
```
